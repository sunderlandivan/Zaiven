Import & pricing

Import inventory

Each spreadsheet row becomes one inventory line (same product name on multiple rows stays separate). Re-importing replaces the list; item ids use the row position (row_0, row_1, …).

Pricing scan

Rows with a TCG Link column use that URL only to resolve product and market price (no name search). Rows without a link still use marketplace search. Full refreshes can take several minutes for large inventories — the status line updates when finished.

Data reset tools

Use these only when needed. Inventory reset clears imported rows and sell workflow status. Orders reset clears order history only.
